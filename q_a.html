Question-1: What are the different ways to select an element in the DOM?

Answer : Selecting an element in the DOM can be done using various methods such as document.getElementById(), document.getElementsByClassName(), document.getElementsByTagName(), document.querySelector(), and document.querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML is used to set or retrieve the HTML content of an element. innerText is used to set or retrieve the text content of an element. textContent is used to set or retrieve the text content of an element, including the text within the element's child elements.

Question-3: What is event delegation in the DOM?

Answer: Event delegation is a technique in JavaScript that allows you to attach a single event listener to a parent element and then delegate the event to its child elements. This reduces the number of event listeners and improves performance.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling is a mechanism in which events propagate up the DOM tree from the target element to its parent elements. When an event occurs on an element, the event will be triggered on that element and then on its parent, and so on, up to the document object.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: Creating, adding, and removing elements using JavaScript can be done using various methods such as document.createElement(), document.appendChild(), document.removeChild(), and document.replaceChild().
